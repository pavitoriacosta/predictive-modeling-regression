# Modelagem Preditiva com Regressão
Predictive modeling project using regression techniques, focusing on data analysis and model interpretation.

##  Descrição do Projeto

Este projeto tem como objetivo desenvolver um modelo de regressão para análise e previsão de uma variável alvo, utilizando técnicas de análise exploratória de dados, pré-processamento e avaliação de desempenho.

O foco está na compreensão dos dados, interpretação dos resultados e construção de um modelo estatístico consistente.

---

##  Etapas do Projeto

- Análise exploratória dos dados (EDA)
- Tratamento e preparação da base
- Construção do modelo de regressão
- Avaliação de desempenho
- Interpretação dos coeficientes

---

## Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

---

## Resultados

O modelo foi avaliado utilizando métricas de regressão, permitindo analisar sua capacidade de explicar a variável alvo e identificar relações relevantes entre as variáveis.
